Bienvenido a la página de Información

A continuación, te presentamos una guía paso a paso para usar la aplicación **RegistrAPP**.

  1. Ingresa a la aplicación: Localiza el icono de **RegistrAPP** en tu dispositivo y ábrela.
  2. Inicia sesión: Ingresa tu nombre de usuario y contraseña.
    • Si olvidaste tu contraseña, selecciona el botón Recuperar.
  3. Opciones al ingresar: Ingresa tu nombre de usuario para habilitar las siguientes opciones:
    • Ingresa tu correo electrónico y haz clic en Aceptar. Recibirás un correo con instrucciones para recuperar tu contraseña.
    • Si necesitas acceso temporal, selecciona Solicitar Contraseña. Esto te proporcionará un código temporal.
    • Para contactar con soporte, selecciona Contactar Soporte, ingresa los datos requeridos y haz clic en Enviar Solicitud.
    • Selecciona tu rol: Docente o Alumno según corresponda.
    • Para escanear un código QR, selecciona el ícono de escaneo en la parte inferior de la pantalla. Esto abrirá la cámara para realizar el escaneo.
